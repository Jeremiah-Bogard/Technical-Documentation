What is HTML

HTML (Hypertext Markup Language) is not a programming language. It is a markup language that tells web browsers how to structure the web pages you visit. It can be as complicated or as simple as the web developer wants it to be. HTML consists of a series of elements, which you use to enclose, wrap, or mark up different parts of content to make it appear or act in a certain way. The enclosing tags can make content into a hyperlink to connect to another page, italicize words, and so on. For example, consider the following line of text:

Hello World

This text was generated by code using the <p> element. The p stands for paragraph. So in html the text would look like this:

<p>Hello World</p>

The Meaning

So I know how to write text but what exactly does it mean? How does it work? What makes it do what is does?

To write code in html we use tags. There are thousand upon thousands of different tags, but we will just stick with the <p> tag for know.

For every paragraph you write you need an opening and a closing tag. In between the tags is the content you wish to display. Let's take our example from earlier. So we have our Hello World text inclosed in some <p> tags:

<p>Hello World</p>

Now what is it doing

  • Opening Tag: The opening tag is the "<p>" part. This bit of code tells the html interperter that we are writing a paragraph.
  • Content: The content is our text "Hello World". This is the content that our paragraph contains. It is telling the html interperter to display this text to the screen.
  • Closing Tag: The closing tag is the "</p>". It ends the paragraph. Basically it explains to the html interperter that the paragraph has ended. And everything between the opening tag and this one needs to be display on the screen.

Nesting Elements

A <p> tag is also known as an element. Of course there are lots of other elements other than the <p> tag. One tag you will most likely use on a regular basis is the <strong> tag. This tag makes the text bold. So it can turn our dog from being very grumpy to:

My dog is very grumpy.

Do you see the difference? By adding the <strong> tag we put emphasis on very. So our final code looks like:

<p>My dog is <strong>very</strong> grumpy.</p>

By taking the <strong> element and adding it inside the <p> element, we created a nest. We nested the <strong> element inside of the <p> element. Now to nest an element inside of another element, your nested elements tags go inside of the original element. In other words do NOT do this:

<p>My dog is <strong>very grumpy</p></strong>

Attributes

There are a ton of attributes for programming. Two of the most commonly used attributes are the class and id attributes. These are useful for styling, in page links, etc. But we will just explore how to add these attributes to our Hello World paragraph.

If you recall from earlier in the website we looked at how to create a paragraph using the <p> tag. To style a paragraph you would use CSS. here is a sample of CSS we could do to style our paragraph.

p { color: black; }

Unfortunatly this will style all of our paragraphs, so how do we only style the one Hello World paragraph? This is where attributes come in. The id attribute will allow us to style only this paragraph. To add an id attribute we write in the opening tag id="attribute_id". For example:

<p id="hello_world">Hello World</p>
<p>My dog is <strong>very</strong> grumpy</p>

Now we can select the id "hello world" in our CSS code to style the one paragraph:

#hello_world { color: red };
Hello World
My dog is very grumpy
